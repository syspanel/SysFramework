PHP Helpers Documentation

1. URL Helper

This function retrieves a URL for a given route.

function url(?string $name = null, $parameters = null, ?array $getParams = null): Url

Usage Example:

url('route.name', ['param1' => 'value1']);

Returns the URL for the specified route.

2. Response Helper

This function retrieves the response object.

function response(): Response

Usage Example:

response()->redirect('route.name');

Used for handling HTTP responses.

3. Request Helper

This function retrieves the request object.

function request(): Request

Usage Example:

request()->getInputHandler();

Useful for accessing request data.

4. Input Helper

Fetches a specific input value or the full input handler object.

function input($index = null, $defaultValue = null, ...$methods)

Usage Example:

$name = input('name');

Retrieves the input data by the specified key.

5. CSRF Token Helper

Retrieves the current CSRF token.

function csrf_token(): ?string

Usage Example:

$token = csrf_token();

Useful for adding CSRF protection to forms.

6. Asset URL Generator

Generates the URL for a static asset (CSS, JS, images, etc.).

function asset($path)

Usage Example:

asset('css/style.css');

Returns the full URL to the asset file.

7. Sanitize Input Data

Cleans input data to prevent XSS and other security risks.

function sanitize($data)

Usage Example:

$cleanInput = sanitize($userInput);

Strips potentially harmful characters from input.

8. Log Error

Logs error messages to a specified log file.

function logError($message, $file = 'error.log')

Usage Example:

logError('Database connection failed');

Writes error messages to the error.log file.

9. Secure Session Start

Starts a secure session with cookie-only settings.

function startSecureSession()

Usage Example:

startSecureSession();

Starts the session if it is not already started.

10. Format Date

Formats a date string into the specified format.

function formatDate($date, $format = 'd/m/Y H:i')

Usage Example:

formatDate('2025-08-30 14:30:00');

Returns the date formatted as 30/08/2025 14:30.

11. Format Currency

Formats a numeric value as currency.

function formatCurrency($amount, $currencySymbol = 'R$', $decimals = 2)

Usage Example:

formatCurrency(2500);

Returns the formatted value as R$ 2.500,00.

12. Autoload Classes

Automatically loads class files.

function autoload($className)

Usage Example:

spl_autoload_register('autoload');

Registers the autoloader for classes.

13. Render View

Loads and renders a view file.

function loadView($viewName, $data = [])

Usage Example:

loadView('header', ['title' => 'Home Page']);

Includes the specified view file.

14. CSRF Token Generation

Generates a CSRF token for session protection.

function generateCsrfToken()

Usage Example:

$token = generateCsrfToken();

Returns a secure CSRF token.

15. Redirect

Redirects the user to a specified URL.

function redirect($url, $statusCode = 302)

Usage Example:

redirect('https://example.com');

Performs an HTTP redirect.
