Apresentação do App PHP Framework

Bem-vindo à documentação do seu framework PHP personalizado!

1. Introdução

Este projeto é um framework PHP personalizado, desenvolvido com uma estrutura robusta e modular. Ele permite o uso de BladeOne para renderização de templates, Illuminate Database para gerenciar o banco de dados, e o Bootstrap 5 para estilização das páginas. O objetivo principal desta aplicação é fornecer uma base sólida para a criação de aplicações web escaláveis e fáceis de manter.

2. Estrutura do Projeto

A estrutura do projeto é organizada da seguinte forma:

  • app/: Contém os controladores e lógica da aplicação.
  • core/: O núcleo do sistema com funcionalidades essenciais.
  • public/: Arquivos acessíveis publicamente, como CSS e JS.
  • resources/: Contém os templates BladeOne e outros recursos.
  • routes/: Definições das rotas da aplicação.
  • database/: Migrações e configurações de banco de dados.
  • config/: Arquivos de configuração da aplicação.
  • vendor/: Dependências instaladas via Composer.

3. Configuração do Ambiente

Para configurar o ambiente, siga os passos abaixo:

  1. Renomeie o arquivo .env.example para .env.
  2. Configure as variáveis de ambiente, como a conexão ao banco de dados, no arquivo .env.
  3. Instale as dependências do projeto executando composer install.

4. Início Rápido

Siga os passos abaixo para iniciar a aplicação localmente:

  1. Execute o comando php -S localhost:8000 -t public/ para rodar o servidor embutido do PHP.
  2. Acesse a aplicação em http://localhost:8000.
  3. Certifique-se de que as permissões de pasta, como storage/ e cache/, estão corretamente configuradas.

5. Rotas Principais

As principais rotas disponíveis na aplicação incluem:

  • /clients: Página para listar, criar, editar e excluir clientes.
  • /products: Gerenciamento de produtos (caso aplicável).

Essas rotas são definidas no arquivo routes/web.php.

6. BladeOne e Bootstrap

O BladeOne é usado como o sistema de templates. Para criar uma nova página, basta adicionar um arquivo BladeOne no diretório resources/views/. A estilização é feita utilizando o Bootstrap 5, garantindo um design moderno e responsivo.

7. Banco de Dados

As migrações do banco de dados estão localizadas no diretório database/migrations. Para criar as tabelas no banco de dados, execute:

php artisan migrate

Certifique-se de que as credenciais de banco de dados no arquivo .env estão corretas.
