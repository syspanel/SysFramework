Tutorial da Classe Validations

Introdução

A classe Validations oferece uma maneira simples e eficiente de validar dados em seu aplicativo PHP. Este tutorial irá guiá-lo através da implementação e uso desta classe em seu projeto, incluindo a integração com o BladeOne.

1. Classe `Validations`

A classe Validations é responsável por validar diferentes tipos de dados e armazenar erros e dados antigos. Abaixo está o código completo da classe:

<?php

namespace Core\Validations;

class Validations
{
    protected static $errors = [];
    protected static $oldData = [];

    public static function required($field, $value, $message = null)
    {
        if (empty($value)) {
            self::$errors[$field] = $message ?? "O campo $field é obrigatório.";
            return false;
        }
        self::$oldData[$field] = $value;
        return true;
    }

    public static function email($field, $value, $message = null)
    {
        if (filter_var($value, FILTER_VALIDATE_EMAIL) === false) {
            self::$errors[$field] = $message ?? "O campo $field deve ser um e-mail válido.";
            return false;
        }
        self::$oldData[$field] = $value;
        return true;
    }

    public static function integer($field, $value, $message = null)
    {
        if (filter_var($value, FILTER_VALIDATE_INT) === false) {
            self::$errors[$field] = $message ?? "O campo $field deve ser um número inteiro.";
            return false;
        }
        self::$oldData[$field] = $value;
        return true;
    }

    public static function minLength($field, $value, $min, $message = null)
    {
        if (strlen($value) < $min) {
            self::$errors[$field] = $message ?? "O campo $field deve ter no mínimo $min caracteres.";
            return false;
        }
        self::$oldData[$field] = $value;
        return true;
    }

    public static function maxLength($field, $value, $max, $message = null)
    {
        if (strlen($value) > $max) {
            self::$errors[$field] = $message ?? "O campo $field deve ter no máximo $max caracteres.";
            return false;
        }
        self::$oldData[$field] = $value;
        return true;
    }

    public static function pattern($field, $value, $pattern, $message = null)
    {
        if (!preg_match($pattern, $value)) {
            self::$errors[$field] = $message ?? "O campo $field não corresponde ao padrão esperado.";
            return false;
        }
        self::$oldData[$field] = $value;
        return true;
    }

    public static function same($field, $value, $expected, $message = null)
    {
        if ($value !== $expected) {
            self::$errors[$field] = $message ?? "O campo $field deve ser igual a $expected.";
            return false;
        }
        self::$oldData[$field] = $value;
        return true;
    }

    public static function phone($field, $value, $message = null)
    {
        if (!preg_match('/^\+?[0-9]{10,15}$/', $value)) {
            self::$errors[$field] = $message ?? "O campo $field deve ser um número de telefone válido.";
            return false;
        }
        self::$oldData[$field] = $value;
        return true;
    }

    public static function url($field, $value, $message = null)
    {
        if (filter_var($value, FILTER_VALIDATE_URL) === false) {
            self::$errors[$field] = $message ?? "O campo $field deve ser uma URL válida.";
            return false;
        }
        self::$oldData[$field] = $value;
        return true;
    }

    public static function check()
    {
        return [
            'errors' => self::$errors,
            'oldData' => self::$oldData,
        ];
    }
}
</code>

2. Processamento do Formulário

A seguir está o código para processar o formulário e usar a classe Validations:

<?php

require 'vendor/autoload.php'; // Inclua o autoload do Composer

use Core\Validations\Validations;
use eftec\bladeone\BladeOne;

$views = __DIR__ . '/views'; // Pasta onde estão seus arquivos Blade
$cache = __DIR__ . '/cache'; // Pasta para cache do Blade
$blade = new BladeOne($views, $cache, BladeOne::MODE_DEBUG);

if ($_SERVER['REQUEST_METHOD'] === 'POST') {
    $result = [
        'errors' => [],
        'oldData' => []
    ];

    Validations::required('nome', $_POST['nome']);
    Validations::email('email', $_POST['email']);
    Validations::minLength('senha', $_POST['senha'], 8);

    $result = Validations::check();

    if (!empty($result['errors'])) {
        echo $blade->run("formulario", $result);
        exit;
    }

    echo "Formulário enviado com sucesso!";
}
</code>
