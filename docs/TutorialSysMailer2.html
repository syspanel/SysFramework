Tutorial SysMailer.php

Envie emails com facilidade usando a Classe SysMailer.php

Instalação e Configuração

Para começar, você precisará incluir a Classe SysMailer.php no seu projeto.

  • Faça o download da Classe SysMailer.php;
  • Cole o arquivo na pasta do seu projeto;
  • Inclua a Classe SysMailer.php no seu arquivo PHP.

Configuração do SMTP

Configure as opções do SMTP para conectar ao servidor de email.

Opção Valor
Host Endereço do servidor SMTP
Porta Número da porta do servidor SMTP
Usuário Nome de usuário do servidor SMTP
Senha Senha do servidor SMTP

Enviando Emails

Use o método send() para enviar emails.

$mailer = new SysMailer([ 'host' => 'seu_smtp', 'port' => 587, 'username' => 'seu_email', 'password' => 'sua_senha', 'encryption' => 'tls', 'from_email' => 'seu_email', 'from_name' => 'Seu Nome', ]); $mailer->send('destinatario@example.com', 'Assunto', 'Corpo do email');

Anexos e Caracteres Especiais

Envie anexos e use caracteres especiais nos emails.

  • Use o método send() com o parâmetro $attachments;
  • Use o método encodeSubject() e encodeBody() para caracteres especiais.

Log de Envio e Erro

Registre os logs de envio e erro para monitorar o sistema.

  • Use o método logSuccess() e logError() para registrar logs;
  • Configure o arquivo de log no construtor da Classe SysMailer.php.
