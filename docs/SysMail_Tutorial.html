SysMail Tutorial

Aprenda a configurar e usar o sistema de envio de e-mails em seu framework PHP

1. O que é o SysMail?

O SysMail é uma classe de envio de e-mails para seu framework PHP, que permite o envio de mensagens por SMTP ou Sendmail. Ele suporta configuração de provedores de e-mail e templates personalizados para mensagens.

2. Configuração no arquivo .env

Antes de usar o SysMail, você precisa configurar as variáveis de ambiente no arquivo .env. Aqui está um exemplo:

MAIL_TRANSPORT=smtp
MAIL_HOST=smtp.example.com
MAIL_PORT=587
MAIL_USERNAME=your_username
MAIL_PASSWORD=your_password
MAIL_ENCRYPTION=tls
MAIL_FROM_ADDRESS=your_email@example.com
MAIL_FROM_NAME=Your Name

3. Classe SysMail

A seguir está o código da classe SysMail:

namespace Core;

use Exception;

class SysMail
{
    protected $config;
    protected $transport;

    public function __construct()
    {
        $this->config = $this->loadConfig();
        $this->setupTransport();
    }

    protected function loadConfig()
    {
        return [
            'transport' => getenv('MAIL_TRANSPORT') ?: 'smtp',
            'host' => getenv('MAIL_HOST') ?: 'localhost',
            'port' => getenv('MAIL_PORT') ?: 587,
            'username' => getenv('MAIL_USERNAME') ?: '',
            'password' => getenv('MAIL_PASSWORD') ?: '',
            'encryption' => getenv('MAIL_ENCRYPTION') ?: 'tls',
            'from' => [
                'address' => getenv('MAIL_FROM_ADDRESS') ?: '',
                'name' => getenv('MAIL_FROM_NAME') ?: ''
            ]
        ];
    }

    // Demais métodos...
}

4. Usando o SysMail

Agora que o SysMail está configurado, você pode utilizá-lo para enviar e-mails. Aqui está um exemplo de uso:

$mailer = new \Core\SysMail();
$templatePath = __DIR__ . '/../views/emails/welcome.php';
$data = ['name' => 'John Doe'];
$emailBody = $mailer->renderTemplate($templatePath, $data);

$mailer->send('recipient@example.com', 'Welcome!', $emailBody);

Certifique-se de que o template de e-mail existe em /views/emails/welcome.php, e que você o está populando com os dados corretos.

5. Métodos e Personalização

A classe SysMail suporta os seguintes métodos para personalização:

  • setupSmtp(): Configura o envio por SMTP
  • setupSendmail(): Configura o envio por Sendmail
  • send(): Envia o e-mail com os parâmetros de destinatário, assunto e corpo
  • renderTemplate(): Renderiza templates de e-mail com dados dinâmicos

6. Conclusão

O SysMail é uma solução robusta para o envio de e-mails em seu framework PHP. Com a capacidade de configurar provedores de e-mail e usar templates personalizados, ele oferece flexibilidade para atender às necessidades da sua aplicação. Certifique-se de configurar corretamente seu .env e ajustar os templates conforme necessário.
