Tutorial sobre Funções em PHP

1. Introdução

Funções são blocos de código reutilizáveis que realizam tarefas específicas. Elas ajudam a organizar o código e a torná-lo mais legível e modular. Em PHP, funções podem ser definidas e chamadas de várias maneiras.

2. Definição de Funções

Para definir uma função em PHP, você usa a palavra-chave function, seguida pelo nome da função, parênteses e chaves. Aqui está a sintaxe básica:

function nomeDaFuncao($param1, $param2) {
        // Código a ser executado
        return $resultado;
}

3. Funções Sem Parâmetros

Você pode definir funções que não aceitam parâmetros. Essas funções podem ser usadas para realizar tarefas sem precisar de informações externas:

function saudacao() {
    echo "Olá, mundo!";
}

saudacao(); // Exibe "Olá, mundo!"

4. Funções com Parâmetros

Funções podem aceitar parâmetros, que são passados entre parênteses na chamada da função. Esses parâmetros podem ser usados dentro da função para realizar cálculos ou manipulações:

function somar($a, $b) {
    return $a + $b;
}

$resultado = somar(5, 10); // $resultado será 15

5. Funções com Valores Padrão

Você pode definir valores padrão para parâmetros. Se nenhum valor for fornecido para esses parâmetros ao chamar a função, o valor padrão será usado:

function cumprimentar($nome = "Visitante") {
    echo "Olá, $nome!";
}

cumprimentar(); // Exibe "Olá, Visitante!"
cumprimentar("João"); // Exibe "Olá, João!"

6. Funções Anônimas

Funções anônimas (ou funções lambda) são funções sem nome que podem ser usadas como valores:

$soma = function($a, $b) {
    return $a + $b;
};

echo $soma(5, 10); // Exibe 15

7. Funções como Argumentos

Você pode passar funções como argumentos para outras funções:

function aplicarFuncao($callback, $valor) {
    return $callback($valor);
}

$resultado = aplicarFuncao(function($x) {
    return $x * 2;
}, 5); // $resultado será 10

8. Escopo de Variáveis em Funções

As variáveis definidas dentro de uma função têm um escopo local, o que significa que elas não podem ser acessadas fora da função:

function exemplo() {
    $variavelLocal = "Eu sou local!";
    echo $variavelLocal;
}

exemplo(); // Exibe "Eu sou local!"
echo $variavelLocal; // Gera um erro, pois $variavelLocal não está definida fora da função

9. Funções Recursivas

Funções recursivas são funções que se chamam a si mesmas. Elas são úteis para resolver problemas que podem ser divididos em subproblemas semelhantes:

function fatorial($n) {
    if ($n <= 1) {
        return 1;
    } else {
        return $n * fatorial($n - 1);
    }
}

echo fatorial(5); // Exibe 120

10. Funções e Tipagem

PHP permite especificar tipos de retorno e tipos de parâmetros para funções, proporcionando maior segurança e clareza:

function multiplicar(float $a, float $b): float {
    return $a * $b;
}

echo multiplicar(4.5, 2); // Exibe 9.0

11. Conclusão

Funções são uma parte essencial do desenvolvimento em PHP, permitindo a criação de código modular, reutilizável e legível. Compreender como definir, usar e manipular funções é crucial para se tornar um desenvolvedor PHP eficiente.
